```text
类加载器：
虚拟机自带的加载器，启动类（根）加载器，application加载器，扩展类加载器
external下面有个rt.jar包，里面有所有的class

双亲委派机制：安全的
APP---》EXC ----》BOOT（最终执行根加载器）
1 类加载器收到类加载的请求
2 将这个请求向上委托给父类加载器去完成，一直向上委托，直到启动类加载器boot
3 启动加载器检查是否能够加载当前这个类，能加载就结束，否则抛出异常，通知子加载器进行加载
4 重复步骤3




```
